# 메뉴 (Menu) 기능 요구사항

## 개요

카페 키오스크에서 판매하는 음료 메뉴의 구성과 특징을 정의합니다. 각 메뉴는 고유한 상품 번호, 이름, 가격 정보를 가집니다. 일부 메뉴는 아이스(ICE) 전용으로 제공되며, 나머지 메뉴는 따뜻한(HOT) 음료와 차가운(ICE) 음료 옵션을 모두 제공합니다.

# 요구사항

- 메뉴 항목은 상품 번호, 상품명, 가격으로 구성된다.
- 플랫화이트와 코코넛 커피는 아이스 전용 메뉴이다.
- 다른 모든 음료는 HOT과 ICE 옵션을 선택할 수 있다.

- 예시 메뉴:
  - 아메리카노, 4000원
  - 카페라떼, 4500원
  - 플랫화이트 (ICE), 4500원
  - 에스프레소, 3500원
  - 솔트커피, 5000원
  - 롱블랙, 4500원
  - 코코넛커피 (ICE), 5500원
  - 에그커피, 5500원
